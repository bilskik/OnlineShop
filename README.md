# ONLINE SHOP

This is a full-stack application developed during the 4th semester at Warsaw University of Technology. The objective is to create a basic online shop.

## Technology

- JavaScript v16.16.0,
- React v18.2.0,
- CSS,
- Spring Boot v3.0.5,
- Spring Data Jpa,
- Hibernate,
- MySQL,
- lombok,
- jwt,
- Spring Security v6.0.0
- jackson
- postman
- logback
